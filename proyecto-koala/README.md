# Proyecto Koala
1. Clonar proyecto
2. ````npm i````
3. Renombrar y configurar archivo ````.env.template```` y renombrarlo a ````.env````

4. Iniciar el modo desarrollo, te mueves a la carpeta de proyecto-koala que es un backend en NEST ````npm run start:dev````
